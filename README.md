# EV Skill Gap Analyzer 🚗⚡

A hackathon-ready project to identify skill gaps in automotive manufacturing and the EV ecosystem.

## 🌍 Problem Statement
The automotive industry is rapidly transitioning to electric vehicles (EVs). Traditional manufacturing skills often don’t align with the new requirements in battery technology, power electronics, embedded systems, and sustainability.  
This project helps identify **workforce skill gaps** and recommends **reskilling pathways**.

## ✨ Features
- Extracts skills from workforce profiles using NLP (spaCy)
- Compares against EV skill ontology
- Identifies missing skills
- Visualizes gaps with interactive charts (Plotly)
- Suggests reskilling courses from online platforms

## 🛠️ Tech Stack
- Python 3.9+
- spaCy (NLP)
- pandas (data analysis)
- plotly (visualization)
